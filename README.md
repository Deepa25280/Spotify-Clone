# Spotify-Clone

A simple Spotify homepage clone built using only HTML and CSS. This project recreates the basic layout and design of Spotify's landing page to practice frontend web development skills.

🚀 Features
* Responsive navigation bar
* Hero section with Spotify-inspired design
* Music cards and playlists section
* Clean and modern UI
* Built entirely with HTML and CSS
* Beginner-friendly project

🛠️ Technologies Used
* HTML5
* CSS3

🎯 Purpose

* This project was created to improve understanding of:

* HTML page structure
* CSS styling and layouts
* Flexbox and Grid
* Responsive web design
* UI cloning and frontend development
